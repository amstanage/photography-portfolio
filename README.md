# LENS/FRAME

Photography portfolio showcasing automotive and architecture work by Nardussy.

## Featured Projects

**Automotive** — BelAir, Eruption Green EcoBoost, Pontiac GTO, Pontiac Le Mans

**Architecture** — Detroit Institute of Art, Michigan Central Station

## Tech Stack

- React 18 (CDN, no build step)
- Babel standalone for JSX transpilation
- Vanilla CSS-in-JS (inline styles)

## Running Locally

Open `index.html` in a browser. No server or build tools required.
